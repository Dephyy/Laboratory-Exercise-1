BSU Secure Check

This project is a C# desktop application that integrates with a SQL database to manage student information and schedules. The system features a login and authentication system to ensure secure access for students and administrators. Once logged in, students can view their personalized schedules, while administrators can manage and verify schedules for all students. The database stores user credentials, class schedules, and student details, enabling efficient student schedule verification and updates. The application also provides error handling, schedule conflict alerts, and secure data management through SQL queries and procedures.
